# Maison Noir — E-Commerce Product Page

> A modern, minimal e-commerce product page with image 
> gallery, size selector, cart functionality, and a smooth 
> checkout flow.

🔗 **Live demo:** [https://maison-noir-luxury-fashion-e-commer.vercel.app/]

---

## Overview

Maison Noir is a single-product e-commerce UI built to 
replicate the feel of a premium fashion brand storefront. 
The focus was on clean visual design, smooth interactions, 
and a complete purchase flow from product browsing to 
order confirmation.

---

## Features

- **Product image gallery** — multiple images with thumbnail navigation
- **Size selector** — interactive picker with in-stock and out-of-stock states
- **Add to cart** — cart state managed with React hooks
- **Cart drawer** — slide-in panel showing items, quantities, and total
- **Quantity controls** — increment/decrement with live price updates
- **Checkout flow** — order summary and confirmation screen
- **Fully responsive** — mobile, tablet, and desktop

## Key Implementation Details

- Cart state managed with useState and useReducer — 
  no external state library needed
- TypeScript interfaces for product data, cart items, 
  and size variants
- Tailwind transition utilities for the slide-in 
  cart drawer animation
- Mobile-first responsive layout
